## 🔧 BUILD PROCESS

### What Happens When You Run `npm run build`

```
src/App.jsx (React Component)
    ↓
Babel (Converts JSX → JavaScript)
    ↓
Vite (Bundles & optimizes)
    ↓
dist/ (Production-ready files)
    ├── index.html
    ├── assets/
    │   └── index-[hash].js
```

### What Happens When You Deploy to Vercel

```
1. Vercel reads vercel.json
2. Runs: npm install
3. Runs: npm run build
4. Takes everything in dist/
5. Hosts on CDN at muthu-portfolio.vercel.app
6. Your site is live! ✅
```
